# AI Weather Agent

This project demonstrates a simple Agentic AI workflow using LangChain and Ollama.

## Features

- AI Agent
- Tool Calling
- Ollama Integration
- Local LLM
- LangChain Workflow

## Run Project

```bash
python3 main.py

## RAG Pipeline

- Semantic similarity search

- LangChain

- HuggingFace Embeddings
